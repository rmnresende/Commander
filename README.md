# Commander API

Este repositório é uma implementação do vídeo/tutorial [.NET Core 3.1 MVC REST API - Full Course](https://www.youtube.com/watch?v=fmvcAzHpsk8&t=10838s),
que como o nome sugere é um exemplo básico de implementação de uma API Rest utilizando o .Net Core.

## Quais as ferramentas/frameworks utilizadas(os)?

 * .Net Core 3.1
 * Entity Framework 3.1.8
 * Microsoft SQL Server 2019 (RTM-CU7)] [(via container)](https://hub.docker.com/_/microsoft-mssql-server)
 * NewtonsoftJson 3.1.8
